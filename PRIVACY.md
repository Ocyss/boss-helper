# 隐私政策 - boss-helper-直聘AI求职助手

最后更新：2026年3月

本扩展不自动收集任何个人信息。

我们收集/处理的数据：

- 用户自愿输入的 API Base URL 和 API Key：仅用于调用用户指定的 AI 服务，本地 storage 保存，不上传服务器。
- 用户自愿保存的招聘平台 Cookie：仅用于账号切换功能，本地 storage 保存，不上传、不共享。
- 版本检查：启动时向自家服务器请求版本信息（无个人信息，仅记录IP进行日活统计）。

数据不用于其他目的，不出售、不分享给第三方（除用户指定的 AI 服务 API 调用外）。

用户控制：

- 可随时在扩展选项中删除/修改 API Key 或 Cookie。
- 卸载扩展即删除所有本地数据。

无其他浏览活动收集。扩展仅在 zhipin.com 域名注入脚本。
